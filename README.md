Projeto de criação de Grupos de Aviso:

A API de Grupos de aviso será uma API RestFull construinda totalmente em Java Spring boot. Seu FrontEnd será construido com Angular.

Háverá a possibilidade de:
Criar um novo grupo de aviso com o método Post;;
Atualizar um grupo de aviso através de seu ID pelo método Put;
Deletar um grupo de aviso através de seu ID pelo método Delete;
Buscar um grupo de aviso pelo seu ID com o método Get.
.

O usuário poderá criar um novo grupo de aviso da seguinte forma:
 - Criando um grupo com nome e adicionando um aviso, onde PODERÃO ser adicionados a este grupo (ou seja, a partir daqui é opcional) Empresas, Areas, PerfisMBS, Emails Avulsos e Usuários já cadastrados em sistema.
